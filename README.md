# bitQL

#### 介绍
BitQL是一个内存检索引擎，设计理念是数据存储即索引。首先如果是boolean型数据，则直接用一个bit向量，N个数据，在内存中存放，最多浪费空间不超过8个byte(一个long整形的空间)。其他数据类型都是由一系列的相同维度的bit向量组成。例如把long看出由64个bit向量组成。
